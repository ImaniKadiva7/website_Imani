In this course, you will learn how to develop Concurrent and Bare Metal embedded system prototype using the STM32 micro-controller.

About this Program


In this course you will dive deep into the following:

  • Controlling and running multiple processes on a Single Micro-controller

  • Writing Concurrent, Non-Blocking Code Modules

  • Controlling Digital Pins of the STM32 Micro-controller in a Bare-Metal Fashion.


Real World Application


A real-world application for developing code in C that involves concurrent processing on an STM32 microcontroller in a non-blocking fashion, without using an RTOS, is like building a wireless remote control for a robot or a drone. In such a project, you would have multiple code modules running concurrently to manage various aspects of the system.


In this real-world application, you have multiple code modules running concurrently, allowing the robot or drone to perform tasks like sensing its environment, making decisions, and responding to user input—all while maintaining a non-blocking, cooperative multitasking approach, and without the use of an RTOS.
