How to design and develop a 7-Segment display project and debouncing a switch in a non blocking fashion using bare metal approach and stm32 mcu.

About this Program

Without using HAL library, through this simple course you will learn the following:

  • Interface a 7-segment LED display with shift register and stm32 mcu using Bare Metal approach.

  • Debouncing a switch in a non blocking fashion.

  • Dealing with situation where there are insufficient ports

  • Driving LATCH, DATA and CLOCK lines to write data to shift register.
