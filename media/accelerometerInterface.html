Writing Firmware to communicate STM32 ARM processor and MPU_6050 accelerometer using Wire Library and Bare Metal Approach.

About this Program

Through this simple course you will learn basics of I2C interfacing using wire library on the STM32 platform, writing firmware driving and interrogating I2C device i.e accelerometer and interpreting data from the accelerometer.


The system developed in this program is related to a typical smartphone scenario. When the phone screen is rotated such that they appear in landscape form with correct orientation, similarly to potrait form, the contents appear in potrait form.


In this project we use the small breadboard to which the accelerometer is placed. When the breadboard is rotated to landscape orientation, letter L is displayed on the 7 segment, letter F is displayed when breadboard is flat and letter U is displayed when breadboard rotated upside down.
