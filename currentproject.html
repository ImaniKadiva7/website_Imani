Real-Time Embedded System Prototype

This prototype demonstrates real-time multitasking on a microcontroller, achieving responsive and non-blocking execution across multiple modules.

  1. Concurrent non-blocking task execution
  2. Bare-metal ATmega328P GPIO control
  3. I2C communication with MPU6050
  4. Debounced pushbutton handling

System Modes

  1. Traffic lights + static display
  2. Traffic lights + counter module
  3. Traffic lights + orientation detection

Hardware Used

  1. ATmega328P
  2. MPU6050
  3. Shift Register
  4. 7-Segment Display
  5. LEDs, buttons, resistors
